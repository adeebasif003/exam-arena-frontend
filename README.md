# Project Overview

This project is built using React, TypeScript, Vite, Tailwind CSS, and shadcn-ui.
It is structured for fast development, efficient builds, and clean component-based UI development.

This repository contains all the source code for the application, including frontend components, routing, UI elements, and utility functions. The project is optimized for local development and easy deployment.

# How to Run the Project Locally

To work on the project using your preferred IDE (VS Code, IntelliJ, etc.), follow the steps below.

Prerequisites

Make sure you have the following installed:

Node.js & npm

Recommended: Install Node.js using nvm → https://github.com/nvm-sh/nvm#installing-and-updating

# Setup Instructions
    # 1. Clone the repository
    git clone <YOUR_GIT_URL>

    # 2. Navigate into the project folder
    cd <YOUR_PROJECT_NAME>

    # 3. Install all dependencies
    npm install

    # 4. Start the development server with hot reload
    npm run dev


The development server will start automatically and provide a local preview.

# Deployment

The project can be deployed using any platform that supports static frontend hosting, such as:

~Vercel

~Netlify

~GitHub Pages

~Cloudflare Pages

Build command:

npm run build


This generates the production-ready output in the /dist folder.

Upload the dist folder to your hosting service.

# Custom Domain Setup

If you're hosting on Vercel, Netlify, or Cloudflare, you can easily attach a custom domain.

General steps:

1.Add your domain inside your hosting dashboard

2.Update DNS records (usually CNAME or A record)

3.Wait for DNS propagation

4.Your custom domain is ready!
